Class Hierarchy

  • level.Camera: This class is used to get different views into the level.
  • entity.Entity: Base Class upon which the rest of the ingame objects are based.
    • entity.Agent: Extension of the Entity class.
    • entity.Projectile: Class representing projectiles fired by Agents
    • entity.Star: The star's representation.
  • level.Level: Contains all data related to the level
  • states.State: Base class for all game "states"
    • states.GameState: The controller for the game
    • states.MenuState: The controller for the main menu
    • states.PauseState: The controller for the pause menu
